# 云成本优化竞品定价分析

## 主要竞品

### 1. CloudHealth (VMware)
- **定价**: 按管理资源数量收费，基础版 $100/月
- **优势**: 多云支持，企业级功能
- **劣势**: 复杂，对小团队不友好

### 2. CloudCheckr
- **定价**: 按账号数量收费，基础版 $250/月
- **优势**: 自动化 remediation
- **劣势**: 价格高，设置复杂

### 3. Spot by NetApp
- **定价**: 按节省金额抽成 10-15%
- **优势**: 按效果付费
- **劣势**: 抽成比例高

### 4. DevOps Lab
- **定价**: 按账号收费，$100-500/月
- **优势**: 自动化 rightsizing
- **劣势**: 仅 AWS

### 5. Moia AI
- **定价**: 免费层 + 按节省抽成
- **优势**: 无代码自动化
- **劣势**: 仅限 AWS

1. **价格**: 比竞品低 90%+ ($9 vs $100+)
2. **简单**: 无复杂配置，开箱即用
3. **开源**: 核心引擎开源，可自托管
4. **透明**: 无隐藏费用，无抽成
5. **专注**: 只做成本优化，不做其他

## 目标市场

- **初创公司**: 1-10 人团队，AWS 账单 $100-1000/月
- **个人开发者**: 边车项目，需要基础分析
- **中小团队**: 5-50 人，多云环境
