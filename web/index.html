常见问题

Free 版有功能限制吗？

Free 版包含所有核心分析功能，限制在于每月 3 次分析和 1 个 AWS 账号。对大多数个人开发者已足够。

你们的"无抽成"是什么意思？

竞品如 Spot 会从你节省的金额中抽成 10-15%。我们完全不收抽成 — 你省了多少钱就全是你的。我们只收固定月费。

可以自托管吗？

当然！核心引擎是开源的（MIT 许可证）。你可以完全自托管，不花一分钱。SaaS 版只是提供托管服务和额外功能。

我的 AWS 密钥安全吗？

自托管版数据完全在你控制下。SaaS 版中，密钥经过 AES-256 加密存储，仅用于读取账单数据，不会执行任何操作。

多久能看到优化建议？

首次扫描 5 分钟内完成。上传 CUR 文件或授权 AWS CLI 后，系统会立即分析并生成优化报告。
